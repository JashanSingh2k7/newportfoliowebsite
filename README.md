## ⚡ Architecture & Features

* **The IDE Aesthetic:** Built with a high-contrast dark mode (`#0d1117`) and custom CSS classes that mimic IDE syntax highlighting (Keywords, Strings, Entities, and Comments).
* **Zero-Dependency Scroll Spying:** Utilizes the native browser `IntersectionObserver` API to track scroll state and dynamically update the active sidebar link, completely bypassing the need for heavy routing libraries like React Router.
* **Information Density:** Designed to present technical architectures, systems-level metrics, and C++ workflows in a single-page, zero-fluff format optimized for technical recruiters.
* **Responsive Layout:** A two-column, `position: sticky` desktop layout that seamlessly collapses into a mobile-friendly stacked view.

## 🛠 Tech Stack

* **Framework:** React.js
* **Styling:** Vanilla CSS (Zero component library bloat)
* **Typography:** [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono)
* **Deployment:** Netlify
